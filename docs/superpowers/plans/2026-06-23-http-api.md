# HTTP API 接口 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增 `POST /api/charts` 端点，让其他应用以 multipart 上传 CSV + 配置，返回该图表分享页 / PNG / SVG / CSV 的绝对 URL。

**Architecture:** 在纯逻辑层 `app/spec.py` 增加 `default_spec` 纯函数（省略 layout 时构造单面板默认配置）；在薄路由 `app/main.py` 抽共享 helper `_create_chart`（生成 id + 存储），供现有表单路由与新 API 路由复用，新增 `create_chart_api` 返回 JSON。复用现有 `parsing` / `spec` / `rendering`，不改其行为。

**Tech Stack:** FastAPI + Uvicorn，pytest + FastAPI `TestClient`，SQLite。

## Global Constraints

- 局域网自部署，API **不鉴权**。
- CSV 以 **multipart 文件上传** 传入。
- 返回 **绝对 URL**，用 `request.url_for(...)` 生成。
- 网页展示时间须显示浏览器本地时间；但本 API 的 `created_at` 是给程序消费的字段，返回 UTC ISO 原文（`datetime.now(timezone.utc).isoformat()`，由 `store.save_chart` 已生成），不在前端展示，无需转换。
- 依赖方向：路由 → 逻辑/数据；纯逻辑层不得反向依赖 Web。
- TDD：先写失败测试再实现。测试在 `tests/`，用 `pytest` 运行。

def default_spec(
    parsed: ParsedCSV,
    title: str,
    x_title: str,
    x_eng: bool = False,
    x_log: bool = False,
) -> ChartSpec:
    """省略 layout 时的单面板默认配置：所有 Y 列指派到唯一面板。"""
    return ChartSpec(
        title=title,
        x_title=x_title,
        x_eng=bool(x_eng),
        x_log=bool(x_log),
        panels=[PanelSpec(parsed.y_labels[0], False, False)],
        assign={label: 0 for label in parsed.y_labels},
    )
```

@app.post("/api/charts")
async def create_chart_api(
    request: Request,
    file: UploadFile = File(...),
    title: str | None = Form(None),
    x_title: str | None = Form(None),
    x_eng: bool = Form(False),
    x_log: bool = Form(False),
    layout: str | None = Form(None),
    store: Storage = Depends(get_storage),
):
    raw = await file.read()
    try:
        parsed = read_csv_bytes(raw)
        resolved_title = title or _filename_stem(file.filename)
        resolved_x_title = x_title or parsed.x_label
        if layout is not None:
            spec = _build_spec(resolved_title, resolved_x_title, x_eng, x_log, layout)
        else:
            spec = default_spec(parsed, resolved_title, resolved_x_title, x_eng, x_log)
        validate_spec(spec, parsed)
    except CSVParseError as err:
        raise HTTPException(status_code=400, detail=str(err))
    chart = _create_chart(store, raw, spec)
    return JSONResponse(
        status_code=201,
        content={
            "id": chart.id,
            "page_url": str(request.url_for("chart_page", chart_id=chart.id)),
            "png_url": str(request.url_for("chart_png", chart_id=chart.id)),
            "svg_url": str(request.url_for("chart_svg", chart_id=chart.id)),
            "csv_url": str(request.url_for("chart_csv", chart_id=chart.id)),
            "created_at": chart.created_at,
        },
    )
```
